Hermes plugin for Pygments
==========================

This provides Pygments with support for highlighting three Hermes formats:

1. Grammar Files (.hgr)
2. Parse Trees (.htree)
3. Abstract Syntax Trees (.hast)

Install
--------

  $ python setup.py install

  Or

  $ pip install pygments-hermes
